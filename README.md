<h1 id="como-executar"> Como posso executar o projeto?</h1> 
    
```bash
# Clone o repositório
$ git clone https://github.com/Alexandrehideki13/DekiLetMeAsk.git

# Entre no repositório
$ cd letmeask

# Instale as dependências
$ yarn install

# Execute a aplicação
$ yarn start
```
